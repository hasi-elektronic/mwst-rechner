# Hasi MwSt & Kalkulations-Rechner

MwSt-Rechner für schnelle Berechnungen — Hasi Elektronic.

## Features

- **Kalkulation Verkaufspreis** — Einkaufspreis + Aufschlag % → VK Netto + VK Brutto
- **Netto → Brutto** — 19% MwSt Umrechnung
- **Brutto → Netto** — 19% MwSt Umrechnung
- **Eingebauter Taschenrechner**
- **Click-to-Copy** für alle Ergebnisse
- **Click-to-Übernehmen** vom Verkaufspreis in die Umrechnung

## Stack

- Vanilla HTML/JS
- Tailwind CSS (CDN)
- Google Fonts (Inter)

## Deploy

Hosted auf Cloudflare Pages.
